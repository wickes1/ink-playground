# Ink Playground

Minimal Ink script editor and runtime.

## Features

- Monaco editor with Ink syntax highlighting
- Interactive story runtime
- Real-time variable tracking
- Example scripts included

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Monaco Editor
- inkjs

## Usage

```bash
pnpm install
pnpm run dev
```

1. Write Ink script in the editor
2. Click Run or press Cmd+Enter
3. Make choices to progress the story

## Examples

- Hello World - Basic introduction
- Choices - Branching narratives
- Variables - State tracking
- Knots - Story structure
- Advanced - Complex features
